# Prática - Qualidade de Software em PHP

## Objetivo
Aplicar conceitos de qualidade de software em um sistema simples, incluindo análise de código, testes automatizados e integração contínua.

## Tecnologias Utilizadas
- PHP
- PHPUnit
- Git e GitHub
- GitHub Actions

## O que foi feito
- Análise estática do código
- Correção de erros (bugs)
- Criação de testes automatizados com PHPUnit
- Configuração de CI com GitHub Actions

## Como executar o projeto
vendor/bin/phpunit

composer install
